Voucher Numbering — {{ fy_tag }}{% if fy_tag == current_fy %} (current){% endif %}

Voucher numbers are per financial year: the FY is part of the number (e.g. {{ fy_tag }}-SAL-000001) and numbering restarts from 1 each year, so the same number can exist in different years. Numbers auto-increment; here you only set the maximum number allowed per FY for each voucher type — the limit is set once and applies to every financial year (each year gets its own full range). Once a year's limit is reached, no more vouchers of that type can be created in that year until an admin increases it (0 or blank = unlimited). Includes Settlement (Matching) vouchers.
